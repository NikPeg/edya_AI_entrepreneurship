# Домашние задания

Здесь хранятся материалы и выполненные домашние задания курса.

## Структура

- `homework_1/` - Домашнее задание 1: Формулирование идеи продукта
- `homework_2/` - Домашнее задание 2 (будет добавлено позже)
- ...

## Формат

Каждое домашнее задание содержит:
- Описание задания (README.md)
- Шаблоны для выполнения
- Выполненное задание (презентация/документы)
